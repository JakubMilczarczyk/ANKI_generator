# ANKI Flashcard Generator

**ANKI Flashcard Generator** is a flagship portfolio project designed to automate the creation of high-quality flashcards compatible with the ANKI learning platform. The tool demonstrates expertise at the intersection of Data Engineering (ETL pipelines), AI Engineering (LLM interaction, data parsing), and MLOps (data and prompt versioning).

The main goal of the project is to provide a reliable, user-friendly solution for generating structured learning materials from raw text data. It showcases modern software engineering practices, including clean code, robust testing, and professional documentation.

This project is intended as a central element of a professional portfolio, highlighting the ability to build complete, production-grade AI solutions. It is also used personally for professional development and exam preparation.

**Key Features:**
- Automated ETL pipeline for transforming text data into structured flashcards.
- Integration with advanced language models to ensure high-quality content.
- Command-line interface for easy and efficient usage.
- Focus on data validation and consistency for reliable learning materials.
- Designed with scalability and best practices in mind, including data and prompt versioning.

The ANKI Flashcard Generator is ideal for learners, educators, and professionals seeking to streamline the process of creating effective study materials.
